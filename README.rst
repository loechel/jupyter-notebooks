Install
=======

.. code:: console

    git clone https://github.com/loechel/jupyter-notebooks.git
    cd jupyter-notebooks
    python3 -m venv .
    source bin/activate
    pip install -r requirements.txt
    jupyter-nbextension install rise --py --sys-prefix
    jupyter-nbextension enable rise --py --sys-prefix
